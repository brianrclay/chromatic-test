# FocusTrapProvider

### Installation

`npm install @kyper/a11y --save`

<PeerDependencies json={packageJson} />

### Overview

If your application needs to know when a `FocusTrap` component has captured and ended the focus control, you can use the `FocusTrapProvider` to wrap your application. This includes support for multiple instances of `FocusTrap` at a time.

### Props
<Props of={FocusTrapProvider} />

### Usage

```
import { FocusTrapProvider } from '@kyper/a11y'

class App extends React.Component {
  _onFocusTrapStart = () => {
    console.log('Trap Start')
  }

  _onFocusTrapEnd = () => {
    console.log('Trap End')

  }

  render() {
    <FocusTrapProvider onEnd={_onFocusTrapEnd} onStart={this._onFocusTrapStart}>
      <SomeChildWithFocus />
    </FocusTrapProvider>
  }
}
```
